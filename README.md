# Reservation Room System (Django)

## Setup

```bash
pip install -r requirements.txt
python manage.py migrate
```

## Run

```bash
python manage.py runserver
```

- **Seat Reservation:** http://127.0.0.1:8000/
- **Meeting Room Reservation:** http://127.0.0.1:8000/meeting-rooms/


Create a superuser and manage data:

```bash
python manage.py createsuperuser
```
